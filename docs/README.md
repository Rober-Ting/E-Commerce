## 💡 贡献指南

### 添加新文档

1. 确定文档类型（参考 [文档组织指南](DOCUMENTATION_GUIDE.md)）
2. 选择合适的目录
3. 使用规范的文件名
4. 添加清晰的标题和目录
5. 更新本索引文档

### 文档命名规范

```
✅ GOOD:
- API_TESTING_GUIDE.md
- DEBUG_LOGIN_GUIDE.md
- PHASE2_PROGRESS.md

❌ BAD:
- guide.md
- test.md
- doc1.md
```

详见：[文档组织指南](DOCUMENTATION_GUIDE.md)
